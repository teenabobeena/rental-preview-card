### What I learned

I learned how to select several classes with one CSS selector:

```css
#sedan button:hover,
#suv button:hover,
#luxury button:hover {
    background-color: inherit;
    color: inherit;
    cursor: pointer;
}
```
Also I learned how to use 'inherit' to make a button change to the color of the parent element on hover.
